# Simple Todo App - Graph 示例

这是一个使用 Graph-Driven Development 构建的简单 Todo 应用的示例。

## 图结构说明

```
用户认证 ──────────────────────────────────────────┐
      │                                           │
      │ depends_on                                │ depends_on
      ▼                                           │
任务服务 ──────────► 任务模型                      │
      │                                           │
      │ depends_on                                │
      ▼                                           │
 API 路由 ─────────────────────────────────────────┤
      │                                           │
      │ depends_on                                │
      ├──────────────► 任务列表组件                │
      │                    │                     │
      │                    │ contains            │
      │                    ▼                     │
      │                 首页 ◄────────────────────┘
      │                    │
      │                    │ contains
      │                    ▼
      └──────────────► 任务表单组件
```

## 模块说明

| 模块 | 描述 | 预估代码行数 |
|------|------|-------------|
| 用户认证 | 处理用户登录、注册、会话管理 | 200 |
| 任务模型 | 任务数据模型定义 | 100 |
| 任务服务 | 任务 CRUD 操作 | 300 |
| API 路由 | 任务相关的 API 接口 | 150 |
| 任务列表组件 | 展示任务列表的 UI 组件 | 200 |
| 任务表单组件 | 添加/编辑任务的表单 | 150 |
| 首页 | 应用首页，整合所有组件 | 100 |

**总预估代码行数: 1200 行**

## 使用方法

1. 在 Graph-Driven Development 中打开 `graph.json`
2. 可以编辑节点、调整依赖关系
3. 点击"生成代码"按钮，按拓扑序并行生成代码

## 技术栈

- **框架**: Next.js 14 (App Router)
- **认证**: NextAuth.js
- **数据库**: PostgreSQL + TypeORM
- **前端**: React 18 + TypeScript
